ZelCore v2.2.7 changes

Features

  • VeriBlock (VBK) is now available
  • CruxPay is now avaialble! Create your zel.crux CruxID and send funds easily to human-readable name
  • Portuguese language is now available
  • Adds possibility to customize shielding in Full Node
  • Ravencoin Asset list updated
  • Following Full Node binaries updated: Zcash, Axe, Zcoin, ZelCash
  • Neo network node list updated

ZelCore v2.2.6 changes

Features

  • Czech language is now available
  • Adds possibility to edit ZelNode
  • Adds possibility to delete ZelNode
  • Adds checks if Full Node is synced and notifies about status
  • Full Node now shows block date
  • Following Full Node binaries updated: ZelCash, Bitcoin Private, Horizen, Sinovate

Bugs fixes

  • Fixes bug sending large amounts on some ERC20 tokens
  • Fixes backup of all addresses on ZelNodes setup guide
  • Corrects parsing on Zel protocol

ZelCore v2.2.5 changes

Features

  • FullNode changes: ZelCore now has much better checks for running daemon process
  • FullNode changes: Address labels are now available
  • FullNode changes: Maximum button for multiple receivers is now available on Zcash type coins
  • FullNode changes: Better logging and displaying of errors
  • FullNode changes: Added better checks for ip address in ZelNodes
  • FullNode changes: Simplifies and corrects ZelNode setup and its configuration file
  • Adds information about minimal claimable rewards
  • Ravencoin asset list updated
  • FullNode binaries updated for: BTC, ZEC, XSG, SAFE
